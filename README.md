## Event Consumer spring boot starter

Provides the automatic consumer configuration with kafka.

#### Technologies

- [Spring Boot 2.2.3.RELEASE](https://spring.io/projects/spring-boot)
- [Spring Kafka 2.3.5.RELEASE](https://spring.io/projects/spring-kafka)
- [Java 13](https://www.oracle.com/java/technologies/javase-jdk13-downloads.html)
- [Lombok 1.18.8](https://projectlombok.org/)
- [Gradle 6.0](https://gradle.org/)
- [Avro 1.9.2](https://avro.apache.org/docs/1.9.2/)
- [TestContainers](https://www.testcontainers.org/)

#### Compatibility
* [Java 13](https://www.oracle.com/java/technologies/javase-jdk13-downloads.html)
* [Gradle 6.0](https://gradle.org/)
